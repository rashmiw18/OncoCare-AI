# OncoCare AI

OncoCare AI is an AI-powered cancer patient health monitoring platform built using Flask, MySQL, Machine Learning, and Gemini AI.

## Features

- Patient registration and login
- Daily health data tracking
- Symptom tracking
- AI-assisted health insights
- Machine learning anomaly detection
- Health risk alerts
- Interactive health dashboard
- Doctor dashboard
- Health reports
- Gemini AI integration

## Technology Stack

### Backend

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login

### Database

- MySQL

### Artificial Intelligence

- Google Gemini API
- Google GenAI SDK

### Machine Learning

- Scikit-learn
- Isolation Forest
- NumPy
- Pandas

### Frontend

- HTML
- CSS
- JavaScript
- Chart.js
